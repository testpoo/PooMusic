# 铺音乐播放器 / PooMusic

一个简单的 Gtk 音乐播放器。

### 依赖
```
gstreamer1.0-plugins-good  gstreamer1.0-plugins-ugly gir1.2-gst-plugins-base-1.0 gir1.2-gstreamer-1.0 libgssdp-1.6-0 libgstreamer-plugins-bad1.0-0 libgupnp-1.6-0 libgupnp-igd-1.6-0 ibnice10 libva-drm2 libva2 python3-gst-1.0 python3-typing-extensions
```

### 运行
```
python3 PooMusic.py
```
